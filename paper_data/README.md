# Paper Data

### Directory Structure

```
Emergenet
├── all_data: contains a copy of all data used in the paper, from NCBI and GISAID databases
├── enet_predictions: using Emergenet to give seasonal Influenza vaccine recommendations and compares them the WHO recommendations
└── irat_enet: using Emergenet to evaluate emergence and impact risk to strains, akin to CDC's IRAT
```
