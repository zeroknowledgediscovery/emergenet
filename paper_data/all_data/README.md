# All Data

### Directory Structure

```
all_data
├── README.md
├── gisaid_data : sequence data from GISAID database
├── ncbi_data : sequence data from NCBI database
├── gisaid_metadata_ha.csv : name, subtype, accession number for GISAID HA sequences
├── gisaid_metadata_na.csv : name, subtype, accession number for GISAID NA sequences
├── ncbi_metadata_ha.csv : name, subtype, accession number for NCBI HA sequences
├── ncbi_metadata_na.csv : name, subtype, accession number for NCBI HA sequences
├── seq_metadata.xlsx : name, subtype, accession number for all sequences
├── total_sequences.csv : summary of total sequences of each subtype from each database
├── total_sequences.ipynb : computes total_sequences.csv
└── total_sequences.tex : tex table of total_sequences.csv
```
